Readme for FEUP-8

--------------------------------------------------------------------------------
Some important points about FEUP-8:

- To start FEUP-8, just click on the "FEUP-8" shortcut.
- All 3 files (FEUP-8, libfeup8.so and conf.ini) must be on the same directory
- The shortcut will not work if the folder has spaces in its name
- For those who are more tech-savvy, another way to start FEUP-8 is by
executing on a terminal "./libfeup8.so"
- If you use the shortcut, you may be prompted with a box asking if you trust
 FEUP-8 to be executed. Please click on the trust button.
- The server address and port for the online component may be edited on conf.ini
- To enter the "@" character, click on Shift + 2
